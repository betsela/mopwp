This plugin adds extra features for woocommerce to help improve your online shops. Add swatches for variations, custom thumbnail gallery's, custom size charts and tweak variation pricing and all your add to cart buttons.

== Description ==

Plugin with extras for woocommerce.

== Installation ==

Install the plugin into the `/wp-content/plugins/` folder, and activate it.
